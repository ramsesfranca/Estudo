# Estudo

Projeto para estudar novas tecnologias 

## Techs

- ASP.NET Core 5
- ASP.NET WebApi Core with JWT Bearer Authentication (Falta)
- Entity Framework Core 5
- AutoMapper
- Swagger UI with JWT support

- SOLID and Clean Code
- DDD
- Unit of Work
- Repository
